TAI SAO CO THU MUC main/ ?
===========================
Maven phan chia ro rang 2 loai code:

  main/  -> Code san pham (chay tren server thuc)
  test/  -> Code kiem thu (chi chay khi test, khong deploy)

Thu muc main/ chua 2 thu muc con:
  java/      -> Tat ca file .java (logic he thong)
  resources/ -> Tat ca file cau hinh: application.properties, SQL...

Ly do tach java/ va resources/:
  Maven xu ly chung khac nhau.
  java/      -> Duoc bien dich (compile) thanh bytecode .class
  resources/ -> Duoc sao chep nguyen xi vao trong file .jar
