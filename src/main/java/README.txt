THU MUC: src/main/java/
========================
Chua toan bo file .java cua ung dung - Maven se compile thu muc nay.

Tiep tuc phan cap theo package:
  java/com/wms/    Package goc cua du an

TAI SAO PACKAGE LA com/wms/ MA KHONG PHAI CHI LA wms/?
  Quy uoc Java tu nam 1995: ten package = ten mien dao nguoc
  com   = Commercial (to chuc thuong mai)
  wms   = Workspace Management System (ten du an)
  -> Dung de tranh trung ten voi thu vien khac
  -> VD: com.google.gson, org.springframework.core, com.wms.dao

Ten thu muc PHAI KHOP voi khai bao package trong file Java:
  Thu muc: java/com/wms/dao/
  File:    SpaceDAO.java
  Dong 1:  package com.wms.dao;   <- phai khop 100%
