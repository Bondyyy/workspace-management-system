TAI SAO PACKAGE LA com/wms/ MA KHONG PHAI chi la wms/ ?
=========================================================
Day la QUY UOC DAT TEN PACKAGE cua Java (ton tai tu nam 1995).

Cau truc: [to-chuc].[ten-du-an].[module]
  com   = Commercial (to chuc thuong mai)
  wms   = Workspace Management System (ten du an)

Tai sao can tien to "com"?
  Tranh xung dot ten voi cac thu vien khac.
  Vi du: neu ban dat package la "wms.auth" va co 1 thu vien nao do cung
  dat ten "wms.auth" -> Java khong biet dung cai nao.
  Voi "com.wms.auth" -> unique, khong xung dot.

Trong thuc te cong ty:
  com.facebook.auth
  com.google.api
  org.springframework.security  (org = non-profit)
  -> Moi cong ty co prefix rieng, khong ai trung ai.

WmsApplication.java nam CHINH xac tai day (com/wms/) vi:
  @SpringBootApplication o day -> Spring scan tat ca sub-package ben duoi
  Neu dat sai cho -> Spring khong tim thay cac Controller, Service...
