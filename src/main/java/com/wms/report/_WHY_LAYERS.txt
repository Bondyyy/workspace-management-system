TAI SAO MOI MODULE CO: controller/ service/ repository/ entity/ dto/ ?
=======================================================================
Day la KIEN TRUC PHAN TANG (Layered Architecture) - chuan cong nghiep.
Moi tang chi lam 1 nhiem vu, khong lam viec cua tang khac.

LUONG DU LIEU: Client -> Controller -> Service -> Repository -> Database
                         <-           <-          <-           <-

┌─────────────┬──────────────────────────────────────────────────────┐
│  controller/│ Tiep nhan HTTP request, tra HTTP response            │
│             │ KHONG xu ly logic - chi goi Service roi tra ket qua  │
│             │ Vi du: nhan POST /api/bookings -> goi bookingService  │
├─────────────┼──────────────────────────────────────────────────────┤
│  service/   │ Xu ly toan bo logic nghiep vu                        │
│             │ Tinh toan, kiem tra dieu kien, phoi hop cac module   │
│             │ KHONG biet gi ve HTTP, KHONG query DB truc tiep       │
│             │ Vi du: kiem tra space con trong -> khoa -> tao booking│
├─────────────┼──────────────────────────────────────────────────────┤
│ repository/ │ Giao tiep voi Oracle Database                        │
│             │ Chi co cac ham: findById, save, delete, query...     │
│             │ KHONG co logic gi ca - chi doc/ghi du lieu           │
│             │ Dung Spring Data JPA -> tu sinh SQL, khong can viet  │
├─────────────┼──────────────────────────────────────────────────────┤
│  entity/    │ Anh xa (map) cac bang Oracle thanh Java object        │
│             │1 Entity = 1 bang trong Oracle                        │
│             │ Vi du: Booking.java map voi bang BOOKINGS             │
│             │ @Entity @Table @Column -> Spring tu biet map vao dau  │
├─────────────┼──────────────────────────────────────────────────────┤
│  dto/       │ Data Transfer Object - doi tuong truyen du lieu       │
│             │ KHONG luu vao DB, chi dung de nhan/tra du lieu        │
│             │ TAI SAO KHONG dung Entity truc tiep?                  │
│             │  -> Entity co the chua password, thong tin nhay cam   │
│             │  -> DTO chi tra dung field khach hang duoc phep xem  │
│             │ Vi du: LoginRequest { username, password }            │
│             │        LoginResponse { token } (khong co password)    │
└─────────────┴──────────────────────────────────────────────────────┘

LY DO PHAN TANG - KHONG VIET TAT:
  Neu Service query DB truc tiep -> kho thay doi DB sau nay
  Neu Controller chua logic -> kho test va tai su dung
  Phan tang -> moi nguoi lam 1 tang, it conflict code hon
