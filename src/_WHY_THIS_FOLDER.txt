TAI SAO CO THU MUC src/ ?
==========================
Day la quy uoc bat buoc cua Maven (cong cu build Java).
Maven chi nhan dang va build code neu no nam dung trong src/.

Neu de code ra ngoai src/ -> Maven khong tim thay -> khong build duoc.

Ben trong src/ co 2 thu muc con:
  main/  -> Code chay that su cua ung dung
  test/  -> Code kiem thu (Unit Test), khong chay khi deploy len server

Phan biet nay rat quan trong:
  - Khi build ra file .jar de deploy, Maven chi dong goi code trong main/
  - Code trong test/ chi chay khi goi lenh "mvn test"
